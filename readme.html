Place Saver

Place Saver was made with the following technologies:
HTML5, CSS3, JavaScript
jQuery, jQuery Mobile, Google Maps API v3
PHP, mySQL, JSON, AJAX

The app is simple and easy to use:

  1. Click on an icon to select it.
  2. Click on the map at desired position.
  3. Type the name of your place in the pop up prompt.
  4. Click OK. Your place is now saved locally on your device.
  5. Log in to Metropolia mySQL server and save your place there.
